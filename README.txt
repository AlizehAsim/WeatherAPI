------------
Weather API
------------

*Introduction
This project consists of a REST API that may be called at any given time to provide the user the temperature at his/her live location.

*Requirements
1. Python IDE must be installed

*Files Included:
1. TEST.py
2. README.txt

*Installation
This API can be called as follows:

Launch any Python IDE (PyCharm, Jupyter Notebook, Spyder etc.). The code from TEST.py file must be copied to the IDE and executed.
Click on the localhost URL that is provided in the output to be redirected to your desired Web Page.
